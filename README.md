gengine-samples
===============

Various samples for gengine

#### How to run samples?
Prepare gengine submodule

    git submodule init
    git submodule update

Activate the environment

    source activate
    
Then go to a sample and run it

    cd samples/xxxxx
    gengine-run
    
'gengine-run' will
 * build required dependencies
 * download other binary dependencies
 * build the engine
